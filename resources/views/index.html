Hive Network Status is a small developer API to have Hive, Hive-Engine or HiveSQL network status.

Every endpoints return a response in JSON format.

Hive-Engine nodes

/he/all

Return all nodes, sorted by process time

Query : curl https://netstat.hivelive.me/he/all

/he/best

Return only the fastest Hive-Engine node

Query : curl https://netstat.hivelive.me/he/best

Response example 

{
    "url": "herpc.actifit.io",
    "name": "herpc.actifit.io",
    "online": true,
    "ping": 14,
    "status": {
        "version": "1.10.0",
        "nodeType": "full",
        "chain": "mainnet-hive"
    },
    "lastBlock": {
        "timestamp": "2023-11-03T18:06:36",
        "blockNumber": 32845567,

HiveSQL

/hivesql

Return HiveSQL status

Query : curl https://netstat.hivelive.me/hivesql

Response example 

{
    "status": "online",
    "last_blockchain_block": "79843640",
    "last_database_block": "79843638",
    "last_database_sync": "17:34:27 UTC",
    "sync_gap": "6"
}
Display HiveSQL status
